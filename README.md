# svg-converter# SVG → PNG / JPG 轉換工具

輕量的純瀏覽器 SVG 轉換工具，不需要安裝任何軟體，檔案也不會上傳至任何伺服器。

## 使用方式

1. 下載 `svg-converter.html`
2. 用任何現代瀏覽器直接開啟
3. 拖曳或選擇 SVG 檔案，選好格式與縮放倍率後按下轉換

## 功能

- 支援輸出 **PNG** 或 **JPG** 格式
- 可調整縮放倍率（1× – 4×）
- JPG 可調整輸出品質
- 支援拖曳上傳
- 100% 在瀏覽器端執行，檔案不會離開你的裝置

## 為什麼做這個

HackMD 基於安全考量會擋掉外部 SVG 圖片，需要一個快速把 SVG 轉成 PNG 的方法，讓圖片能在筆記中正常顯示。
